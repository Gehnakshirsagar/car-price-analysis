# car-price-analysis
Exploring what makes car prices go up or down, built with SQL and Python.

## Notebook
Check out the full analysis here: [Vehicle Sales Analysis](https://www.kaggle.com/code/gehnakshirsagar/notebook9f8541efdb)

## What I Did
I explored a dataset of 548,000+ car sales to find patterns in pricing, mileage, and demand.

I wrote SQL queries to answer questions like which brands sell for the most and whether higher mileage cars sell for less. I also cleaned the data by handling missing values and fixing data types, and built 4 charts to visualize the findings. Along the way I ran into some common errors like wrong file paths and deprecated library warnings and fixed them all.

## Key Findings
- Rolls-Royce has the highest average selling price among the top brands
- Cars with low mileage under 20k miles sell for 4x more than high mileage ones
- Sedans are the most sold body type
- Selling prices closely follow market value across all years

## Dataset
Vehicle Sales Data on Kaggle: https://www.kaggle.com/datasets/syedanwarafridi/vehicle-sales-data

Download and place car_prices.csv in the data folder to run the analysis locally.
